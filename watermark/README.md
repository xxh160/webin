## usage

在静态网站服务器上打开 index.html 即可.

不可直接在浏览器中打开, 会出现跨域问题.

会搭建静态网页服务器吗?

如果不会, 在根目录中运行:

```shell
npm install http-server
npx http-server -a 127.0.0.1 -p 5500
```

然后直接访问 127.0.0.1:5500.

或者在 vscode 中装 Live Server 这个插件.
